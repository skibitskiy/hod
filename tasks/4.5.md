# Title
Обновить команду update

# Description
Изменить команду update для работы с JSON файлами.

**Требования от Storage:**
- `storage.exists(id)` — быстрая проверка существования
- `storage.read(id)` — прочитать текущий JSON для мерджа с новыми данными
- `storage.update(id, jsonData)` — атомарная обновление JSON файла

**Входные данные:** ID + CLI опции с новыми значениями
**Выходные данные:** Обновленный JSON файл `{id}.json`

**Новые методы:**
- Прочитать текущий JSON → ParsedTask
- Смёржить новые значения с существующими
- Сериализовать обратно в JSON
- Вызвать `storage.update(id, jsonString)`
