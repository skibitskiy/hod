# Title
Реализация парсера Markdown задач

# Description
Создать парсер для чтения/записи задач в Markdown формате.

## ParserService

```ts
interface ParsedTask {
  id: string;
  title: string;
  description?: string;
  status: string;
  dependencies: string[];  // всегда есть, минимум []
  [key: string]: string | string[] | undefined;
}

interface ParserService {
  parse(id: string, markdown: string): ParsedTask;
  serialize(task: ParsedTask): string;
}
```

## Формат Markdown

```markdown
# Title
Название задачи

# Description
Описание задачи

# Status
pending

# Dependencies
1, 2, 5
```

## Особенности

- `# Dependencies` всегда присутствует (пустой если нет зависимостей)
- `description` может отсутствовать
- Поддержка кастомных полей из конфига
- Многострочные значения поддерживаются
