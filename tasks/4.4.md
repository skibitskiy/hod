# Title
Обновить команду get

# Description
Изменить команду get для чтения JSON файлов вместо Markdown.

**Требования от Storage:**
- `storage.read(id)` — прочитать JSON строку из файла `{id}.json`
- Возврат должен быть JSON строкой для последующего парсинга

**Входные данные:** ID задачи
**Выходные данные:** ParsedTask (парсинг JSON → ParsedTask)

**Новые методы:**
- Распарсить JSON строку в ParsedTask (через новый метод парсера)
- Вывод в markdown формате (для флага --markdown) требует генерацию markdown из JSON
