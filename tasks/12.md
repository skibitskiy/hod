# Title
Команда `hod next` — вычисление следующих задач

# Description
Реализовать команду `hod next`, которая анализирует `tasks.json` и статусы задач,
возвращая список задач, готовых к выполнению.

## Логика работы

1. Читает `tasks/tasks.json` с графом зависимостей
2. Проверяет статус каждой задачи (из markdown файла или отдельного статуса)
3. Задача готова к выполнению, если:
   - Она не завершена (status != completed)
   - Все её зависимости выполнены (status == completed)

## Статус задачи
Статус хранить в секции `# Status` в markdown файле:
- `pending` — не начата
- `in_progress` — в работе
- `completed` — завершена

## Формат вывода
```
Next tasks to work on:
  ✓ 1  [completed] Инициализация проекта
  ✓ 11 [completed] Настройка Vitest

  → 2 [pending]    Создание модуля конфигурации
  → 10 [pending]   Написание тестов для всех модулей
```

## Опции
```bash
hod next              # показать следующие задачи
hod next --all        # показать все задачи с зависимостями
hod next --graph      # визуализировать граф
```
