# Title
Обновить команду add

# Description
Изменить команду add для создания JSON файлов вместо Markdown.

**Требования от Storage:**
- `storage.create(id, jsonData)` — создать задачу с JSON контентом
- `storage.list()` — вернуть список задач для генерации ID (нужны только ID)
- `storage.read(id)` — прочитать JSON для проверки существования родителя
- `storage.exists(id)` — быстрая проверка существования задачи

**Входные данные:** CLI опции → ParsedTask → JSON
**Выходные данные:** JSON файл в `{id}.json`

**Новые методы:**
- Создать JSON из ParsedTask (через новый метод парсера)
- Вызывать `storage.create(id, jsonString)` вместо markdown
