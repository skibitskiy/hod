# Title
Реализовать команду next

# Description
Создать команду next для показа задач, у которых завершены все зависимости. Зависимости считаются выполненными если все задачи имеют статус из nextTaskCriteria в конфиге. Опция --all для показа всех готовых задач. Использует команду get для получения данных задач.
