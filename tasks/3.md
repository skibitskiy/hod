# Title
Реализовать команду get

# Description
Создать команду get для получения задачи по ID. Принимает аргумент id и опциональные названия полей (--title, --dependencies, --status) для указания каких полей вывести. Если поля не указаны — выводит всю задачу в читаемом формате. Форматы вывода: --json для JSON вывода, --markdown для вывода в markdown формате (как в файле).
Поля dependencies и status читаются из index.json
Пример использования:
hod get 1 // выводит задачу с id 1
hod get 1 --title // выводит "1. Заголовок"
