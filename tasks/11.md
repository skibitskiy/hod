# Title
Настройка тестового окружения (Vitest)

# Description
Установить и настроить Vitest для запуска тестов.

## Требования

### Пакеты
- `vitest` — тестовый фреймворк
- `@vitest/ui` — UI для тестов (опционально)
- `typescript` — TypeScript поддержка
- `ts-node` или `tsx` — для запуска TypeScript

### package.json скрипты
```json
{
  "scripts": {
    "test": "vitest",
    "test:ui": "vitest --ui",
    "test:run": "vitest run",
    "test:coverage": "vitest run --coverage"
  }
}
```

### vitest.config.ts
```ts
import { defineConfig } from 'vitest/config';

export default defineConfig({
  test: {
    globals: true,
    include: ['**/*.test.ts'],
    exclude: ['node_modules', 'dist'],
  },
});
```

### Команды CLI
```bash
# Запуск в watch режиме
npm test

# Запуск всех тестов один раз
npm run test:run

# Запуск с UI интерфейсом
npm run test:ui

# Запуск с coverage
npm run test:coverage

# Запуск конкретного файла
npx vitest run path/to/test.test.ts

# Запуск тестов по паттерну
npx vitest run --grep "config"
```

## Структура тестов
```
src/
  config/
    index.ts
    config.test.ts
  storage/
    index.ts
    storage.test.ts
  parser/
    index.ts
    parser.test.ts
```
